# MidiTest

MIDI信号を送るやつです。

## 基本的な使い方

* 起動します
* 出力先を選んでOpenを押します
* チャンネルとかメッセージタイプとかデータバイトを適当にいじくってSendを押します
* 楽しい!(AA略)

## Windows標準のMicrosoft GS Wavetable Synthを使って音を出す

Windowsには標準でMIDI音源が入っているのでそれを使って音を出す事が出来ます。

### データバイト1,2の役割

* データバイト1は音階(0が最低音,127が最高音?)を指定する
* データバイト2は音量(0でミュート,127で最大?)を指定する
